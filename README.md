ROCK-PAPER-SCISSORS GAME

A beginner-friendly command-line implementation of the classic Rock-Paper-Scissors game built using Python. In this game, users compete against the computer across multiple rounds while the program keeps track of scores and determines the overall winner at the end.
This project demonstrates fundamental Python programming concepts through an interactive and engaging game experience.

FEATURES
-> Play Rock-Paper-Scissors against the computer
-> Random computer move generation
-> Input validation to ensure valid user choices
-> Automatic winner determination based on game rules
-> Score tracking across multiple rounds
-> Option to play multiple rounds in a single session
-> Final score summary and overall winner announcement

HOW IT WORKS
The user selects either rock, paper, or scissors, while the computer randomly chooses one of the three options. The program compares both choices according to the standard game rules:

-> Rock beats Scissors
-> Scissors beats Paper
-> Paper beats Rock

After each round, the score is updated and displayed. The user can continue playing additional rounds until they choose to exit the game. At the end of the session, the program displays the final scores and announces the overall winner.

TECHNOLOGIES USED

-> Loops (`while`)

-> Random Module (`random.choice()`)

GETTING STARTED
1. Clone the repository.
2. Ensure Python 3 is installed on your system.
3. Run the program:

python rock_paper_scissors.py

4. Follow the on-screen instructions to play against the computer.
